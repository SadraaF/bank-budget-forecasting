\# Neobank Liquidity Risk Management System üè¶



This is a group project done for a Modern Banking course at AUT. It simulates the operational budget of an Iranian Neobank under hyper-inflationary scenarios using Machine Learning.



\## Features

\- \*\*Synthetic Data Generation:\*\* Simulates 5 years of banking data (Users, USD Rate, Budget) considering the Iranian economic context.

\- \*\*Hybrid ML Model:\*\* Uses Linear Regression (for trend) + XGBoost (for volatility) to predict future costs.

\- \*\*Interactive Stress Testing:\*\* Allows users to adjust Monthly Inflation and User Growth to visualize Liquidity Risks.



\## Tech Stack

\- Python 3.9+

\- Streamlit (UI)

\- Plotly (Interactive Charts)

\- Scikit-Learn \& XGBoost (Machine Learning)
